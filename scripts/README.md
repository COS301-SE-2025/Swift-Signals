# Local Development 

## Prerequisites

- [Docker](https://docs.docer.com/get-docker/) (install Docker Desktop on Windows/Mac)
- [kubectl](https://kubernetes.io/docs/tasks/tools/) (kubernetes CLI)
- [MiniKube](https://minikube.sigs.k8s.io/docs/start/) (local Kubernetes cluster)

Note: Make sure to start Minikube: `minikube start`

## Scripts

- Run `./frontend-dev.sh` to run the frontend locally (On Mac/Linux). 
- Run `./frontend-dev.ps1` to run the frontend locally (On Windows).
